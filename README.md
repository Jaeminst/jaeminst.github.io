
### How to 명령어 'bundle' 을(를) 찾을 수 없습니다.
```bash
$ whereis bundle
bundle: /usr/bin/bundle2.7

$ sudo ln -s /usr/bin/bundle2.7 /usr/bin/bundle
```
